# Netflix Analytics Project (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends in movies and TV shows, including genres, countries, and release patterns.

## Tools & Technologies

* Python 
* Pandas
* NumPy
* Matplotlib / Seaborn

## Dataset

* Netflix dataset containing information about movies and TV shows such as title, genre, country, and release year.

## Key Analysis Performed

* Data Cleaning & Preprocessing
* Movies vs TV Shows comparison
* Genre distribution analysis
* Country-wise content production
* Year-wise trend analysis

## Key Insights

* Majority of content is Movies compared to TV Shows
* Drama and International genres dominate
* Significant growth in content after 2015
* USA and India contribute most content

## How to Run

1. Download the `.ipynb` file
2. Open in Jupyter Notebook
3. Run all cells
